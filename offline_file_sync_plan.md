# Offline File Sync - План разработки

## Обзор проекта
Приложение для синхронизации файлов между Android и ПК через локальную сеть (WiFi) без облачных сервисов.

**Уникальное торговое предложение:** "Синхронизация телефона с компьютером без облака. Настройка за 30 секунд."

**Целевая цена:** $4.99 (Pro версия)

---

## Анализ конкурентов

| Конкурент | Проблемы пользователей |
|-----------|------------------------|
| **Syncthing** | Сложная настройка (20+ минут), "failed elements", дубликаты, высокое потребление RAM |
| **LocalSend** | Не видит устройства, падает на больших файлах (>10GB), нет авто-синхронизации |
| **KDE Connect** | Только для KDE/Linux, много лишнего функционала |
| **Warpinator** | Только Linux, нет авто-синхронизации |

**Наша ниша:** "Syncthing, но простой"

## MVP функционал (v1.0)

### Android приложение
- [ ] Автообнаружение устройств в одной WiFi сети (mDNS/Bonjour)
- [ ] Выбор папки для синхронизации
- [ ] Двусторонняя синхронизация (изменения на оба устройства)
- [ ] Прогресс бар со скоростью и оставшимся временем
- [ ] Работа в фоне (foreground service)
- [ ] Уведомления о завершении/ошибках

### Desktop приложение
- [ ] Windows/macOS/Linux поддержка
- [ ] Минимальный UI (трей-иконка + окно настроек)
- [ ] Выбор папки для синхронизации
- [ ] Подтверждение входящих подключений

### Протокол
- **Обнаружение:** mDNS/Bonjour
- **Передача:** HTTP/2 или raw TCP
- **Безопасность:** TLS для шифрования (опционально в бесплатной версии)

## Техническая архитектура

```
┌─────────────┐      WiFi LAN      ┌─────────────┐
│   Android   │ ◄────────────────► │  Windows/   │
│   (Kotlin)  │   WebSocket/HTTP   │  macOS/     │
│             │   или TCP socket   │  Linux      │
└─────────────┘                    │  (Java/     │
                                   │   Electron) │
                                   └─────────────┘
```

### Стек технологий

**Android:**
- Kotlin
- Coroutines для асинхронности
- WorkManager для фоновой работы
- NanoHTTPD или Ktor для сервера
- jmDNS для обнаружения

**Desktop:**
- Electron (JS) или JavaFX
- Node.js для сетевого слоя
- Bonjour/mDNS для обнаружения

## Риски и решения

| Риск | Решение |
|------|---------|
| Android убивает фоновый процесс | Foreground service + уведомление |
| Разные WiFi сети (гость/основная) | Инструкция пользователю + fallback на QR-код |
| Большие файлы (>10GB) | Потоковая передача, прогресс, resume |
| Конкуренция с бесплатными | Фокус на простоте, не цене |
